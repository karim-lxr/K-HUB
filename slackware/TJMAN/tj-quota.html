Implementing Quota disk rel. gen-2011
--KAR--


Definisce la quota disco utilizzabile da un gruppo utenti o singolo utente attraverso il db aquota.users aquota.group

Procedura di attivazione

  • in /etc/fstab adding, according filesystem you chose: usrquota, grpquota like:
    /dev/VolGroup00/LogVol00     /home    defaults,usrquota,grpquota 1 2

  • activate your quoted filesystem without reboot like:   mount -o remount /home

  • initializing quota DB: cd on fs quoted, case home:   quotacheck -cug /home

  • verify db creation (aquota.users aquota.group) with:   ls -al aquota.*

  • enable quota support without reboot with:   quotaon /home

  • verify if quota supporti is enabled with 'quotaon -p /home'



Quotare l'utente

  • editing and configure quota by each user with: edquota user1

  • assign quota policies inside edquota setting warning-limits(SOFT) and panic-limits(HARD)
    Cave: limits 0 = no limits will be enforced
    limits by bit: 100M => 102.400 200M => 204.800 300M => 307.200 400M => 409.600 500M => 512.000

  • etting graceful time, by days, default 7days with:     edquota -f /home -t

  • reporting: repquota -a for all filesystem,  quota user1 for single user

  • to migrate filesystem remember to erase old aquota.*, repeat this setting steps quota in new filesystem
