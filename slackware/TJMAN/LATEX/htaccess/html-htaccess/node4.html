Direttive .htaccess

Options
Opzioni varie
ErrorDocument
Personalizza la risposta alle condizioni di errore
DefaultType
Definisce il mime predefinito per la directory
AuthName
Nome o descrizione di una zona soggetta ad autenticazione
AuthType
Definizione del tipo di autenticazione (Basic)
require
Dichiarazione degli utenti autorizzati all'autenticazione
order
Ordine di valutazione delle direttive deny e allow
deny
Specifica gli host a cui viene negato l'accesso
allow
Specifica gli host a cui viene concesso l'accesso
